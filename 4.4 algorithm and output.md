4.4 Word Frequency Count

Scenario: Write a program to count the frequency of each word in a paragraph.

Algorithm:

1. Input the paragraph.

2. Split the paragraph into words.

3. Count the frequency of each word using a dictionary.

output:
word frequencies in the paragraph:

python : 3
is:3 
great python:1 
easy:1 
to:1 
learn:1 
and:2
